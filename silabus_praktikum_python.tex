\noindent\textbf{Deskripsi singkat mata kuliah.} Praktikum ini membekali mahasiswa keterampilan Python 3 secara \textbf{bertingkat} (selaras \texttt{sumber\_materi.md} §\,1): \textit{(dasar)} lingkungan pengembangan, sintaks, kontrol alur, dan fungsi; \textit{(menengah)} struktur data, berkas/JSON, modul, \texttt{pip}/\texttt{venv}, penanganan kesalahan, dan OOP dasar; \textit{(lanjut)} \textit{comprehension}/\textit{generator}, pengujian, anotasi tipe dan pemeriksa statis, \textit{logging}, kemasan proyek ringan, serta pengenalan konkurensi; \textit{(jejak sains data)} \texttt{numpy}/\texttt{pandas} pada dataset kecil. \noindent\textbf{Alokasi waktu.} Acuan beban umum KPT untuk praktikum: $\pm$170 menit kegiatan terstruktur per minggu per SKS praktikum per semester. Untuk 1 SKS, prodi dapat merencanakan satu blok $\pm$170 menit/minggu atau pembagian slot lain yang setara; jumlah minggu pada Bagian E dapat dikurangi dengan menggabungkan topik jika beban SKS lebih kecil.

\noindent\textbf{Sarana dan prasarana.} Laboratorium dengan Python 3.10+ (atau versi yang ditetapkan prodi), editor/IDE, akses internet terbatas untuk dokumentasi, proyektor/papan tulis, dan kebijakan K3 serta etika penggunaan perangkat lunak berlisensi.

\subsection*{Penjabaran CPL prodi yang dibebankan pada mata kuliah ini}

\begin{itemize}[leftmargin=*]
  \item \textbf{CPL-1 (Pengetahuan):} Menguasai konsep algoritme dan pemrograman prosedural serta prinsip modularitas, struktur data, penanganan data/berkas dalam konteks Python, dan \textbf{pengenalan} pola kerja kode yang mendekati praktik produksi (tipe statis, \textit{logging}, struktur paket).
  \item \textbf{CPL-2 (Keterampilan umum):} Menerapkan pemikiran logis, sistematis, dan kritis dalam merancang, mengimplementasikan, serta menguji program Python untuk masalah komputasi, termasuk skenario I/O dan tugas ringan \textbf{konkuren/asinkron} pada tingkat pengenalan.
  \item \textbf{CPL-3 (Keterampilan khusus):} Menulis, menguji, dan memperbaiki program Python mulai dari skrip dasar hingga struktur berorientasi objek sederhana, penggunaan pustaka standar, pengujian unit tingkat pengenalan, serta \textbf{pengenalan} manipulasi data numerik/tabular dengan pustaka ilmiah umum.
  \item \textbf{CPL-4 (Sikap):} Menunjukkan tanggung jawab di laboratorium, mematuhi K3 dan etika akademik (lisensi perangkat lunak, kejujuran tugas), serta mampu berkolaborasi saat tugas kelompok.
  \item \textbf{CPL-5 (Pengetahuan / profesionalisme praktis):} Memahami praktik pengelolaan lingkungan virtual, dependensi, kualitas kode (\textit{debugging}, dokumentasi singkat), \textbf{pengenalan} anotasi tipe dan alat pemeriksa, \textit{logging} terstruktur, serta kebijakan kemasan proyek kecil sebagai dasar kerja profesional.
\end{itemize}

\section*{Bagian C. Capaian Pembelajaran Mata Kuliah, Matriks, Strategi, dan Sub-CPMK}

\subsection*{C.1. Capaian pembelajaran mata kuliah (CPMK)}

\begin{itemize}[leftmargin=*]
  \item \textbf{CPMK-1:} Mahasiswa mampu menyiapkan lingkungan Python 3, memahami sintaks dasar, identifier, tipe data, operator, serta masukan/keluaran konsol.
  \item \textbf{CPMK-2:} Mahasiswa mampu merancang alur program dengan percabangan dan perulangan, termasuk pola bercabang dan penggunaan \texttt{match} pada tingkat pengenalan.
  \item \textbf{CPMK-3:} Mahasiswa mampu memecah program menjadi fungsi yang dapat digunakan kembali, memahami parameter, nilai balik, \texttt{*args}/\texttt{**kwargs}, \textit{lambda}, dan anotasi tipe dasar.
  \item \textbf{CPMK-4:} Mahasiswa mampu menggunakan struktur data (\texttt{list}, \texttt{tuple}, \texttt{dict}, \texttt{set}) dan pemrosesan string lanjut untuk manipulasi data.
  \item \textbf{CPMK-5:} Mahasiswa mampu membaca/menulis berkas teks, bekerja dengan JSON sederhana, dan menerapkan penanganan kesalahan dengan \texttt{try}/\texttt{except}/\texttt{else}/\texttt{finally}.
  \item \textbf{CPMK-6:} Mahasiswa mampu mengorganisasi kode dengan modul, mengelola paket menggunakan \texttt{pip}, menggunakan \texttt{venv}, serta melakukan \textit{debugging} dan penerapan gaya kode konsisten.
  \item \textbf{CPMK-7:} Mahasiswa mampu menerapkan konsep lanjut berupa kelas dan pewarisan dasar, \textit{comprehension}/\textit{generator}, serta pengujian unit tingkat pengenalan (\texttt{unittest} atau \texttt{pytest}).
  \item \textbf{CPMK-8:} Mahasiswa mampu pada tingkat \textbf{pengenalan} laboratorium: menerapkan anotasi tipe dan pemeriksaan statis dasar, \textit{logging} terstruktur, struktur paket sederhana berbasis \texttt{pyproject.toml}/\texttt{venv}, serta satu pola eksekusi konkuren ringan (\texttt{concurrent.futures} atau \texttt{asyncio}).
  \item \textbf{CPMK-9:} Mahasiswa mampu memuat dan memanipulasi data numerik/tabular sederhana menggunakan \texttt{numpy} dan \texttt{pandas} (operasi dasar pada satu \textit{dataset} kecil).
\end{itemize}

\subsection*{C.3. Strategi analisis capaian}

Urutan praktikum menggabungkan struktur \textbf{prosedural} (dari program linear ke kontrol alur, fungsi, data, berkas, lalu modul) dan \textbf{hierarkis} (keterampilan dasar menjadi prasyarat tugas berikutnya). Setelah CPMK-6, mahasiswa memasuki \textbf{klaster} OOP, gaya idiomatik (\textit{comprehension}/\textit{generator}), dan pengujian (CPMK-7). \textbf{CPMK-8--9} disisipkan sebagai \textit{blok lanjutan terpadu} (strategi 1 SKS): Sub-CPMK 8.1--8.2 dipadukan dengan minggu 12--13; Sub-CPMK 8.3--8.4 dan 9.1 digabung pada proyek minggu 15 sehingga capaian lanjut tetap terukur tanpa menambah SKS. Rincian topik bertingkat (dasar, menengah, lanjut, dan arah khusus) dijabarkan pada \textbf{C.5} dan diselaraskan dengan \texttt{sumber\_materi.md} §\,1. Pendekatan ini selaras dengan modul praktikum internal Bab 0--8 serta repositori Jupyter acuan.

\vspace{0.25cm}

\noindent\textbf{Matriks pemetaan minggu rencana (Sub-CPMK utama) $\rightarrow$ CPMK.} Lihat kolom Sub-CPMK dan isi bahan kajian pada Bagian E; ringkasan: Minggu 1--2 $\rightarrow$ CPMK-1; 3--5 $\rightarrow$ CPMK-2; 6--7 $\rightarrow$ CPMK-3; 8 $\rightarrow$ CPMK-4; 9 $\rightarrow$ CPMK-5; 10 $\rightarrow$ CPMK-6; 11--13 $\rightarrow$ CPMK-7 dengan benih CPMK-8 (Sub-CPMK 8.1--8.2); Minggu 15 $\rightarrow$ CPMK-8--9 terpadu + proyek; Minggu 14 dan 16 berisi asesmen sumatif.

\subsection*{C.4. Sub-CPMK (indikator operasional)}

\begin{description}
  \item[CPMK-1:] 1.1 Menjalankan Python 3 dan skrip dari terminal/editor/notebook; 1.2 Identifier, variabel, tipe dasar, konversi tipe; 1.3 Operator dan I/O (\texttt{print}, \texttt{input}).
  \item[CPMK-2:] 2.1 Percabangan \texttt{if}/\texttt{elif}/\texttt{else} dan \texttt{match} pengenalan; 2.2 Perulangan \texttt{for}/\texttt{while}, \texttt{break}/\texttt{continue}, bersarang.
  \item[CPMK-3:] 3.1 Definisi fungsi, parameter, nilai balik, ruang lingkup; 3.2 \texttt{*args}/\texttt{**kwargs}, \textit{lambda}, anotasi tipe.
  \item[CPMK-4:] 4.1 Operasi \texttt{list}, \texttt{dict}, \texttt{set}, \texttt{tuple} lanjut; 4.2 String: \textit{slicing}, metode, pemformatan.
  \item[CPMK-5:] 5.1 Berkas teks dan konteks \texttt{with}; 5.2 JSON dan penanganan pengecualian lanjut.
  \item[CPMK-6:] 6.1 \texttt{import}, \texttt{pip}, \texttt{venv}; 6.2 \textit{Debugging} dan konvensi gaya.
  \item[CPMK-7:] 7.1 Kelas, objek, pewarisan dasar; 7.2 \textit{Comprehension} dan \textit{generator}; 7.3 Unit test pengenalan.
  \item[CPMK-8:] 8.1 Menulis anotasi tipe pada fungsi/modul kecil dan menjalankan pemeriksa statis (\texttt{mypy} atau setara); 8.2 Menerapkan \texttt{logging} dengan level dan \textit{handler} sederhana; 8.3 Menyusun paket/proyek lokal dengan \texttt{pyproject.toml} dan \texttt{venv}; 8.4 Menulis contoh minimal konkurensi (\texttt{ThreadPoolExecutor} atau coroutine \texttt{asyncio} sederhana).
  \item[CPMK-9:] 9.1 Memuat \texttt{ndarray} dan \texttt{DataFrame}, seleksi baris/kolom, dan agregasi dasar pada data contoh.
\end{description}
