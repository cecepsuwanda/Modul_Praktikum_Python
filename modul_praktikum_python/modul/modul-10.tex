\section{List dan Dict Comprehension}

\noindent \textit{Comprehension} memadatkan loop pembangun koleksi menjadi satu ekspresi; bentuk set dan generator serupa secara sintaks. Tutorial bab struktur data (\href{https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions}{list comprehensions}, lampiran \textbf{[8]}) dan artikel komunitas \textbf{[19]} memberi variasi pola.

\begin{lstlisting}[caption=Comprehension]
squares = [x * x for x in range(10) if x % 2 == 0]
nama_ke_id = {n.lower(): i for i, n in enumerate(["Ali", "Budi", "Ali"])}
\end{lstlisting}

\section{Generator dengan yield}

\noindent Fungsi yang memuat \texttt{yield} mengembalikan \textit{generator iterator}; nilai dihasilkan \textit{lazy} sehingga hemat memori untuk deret panjang. Lihat juga \href{https://docs.python.org/3/howto/functional.html#generators}{Generators} pada HOWTO fungsional (lampiran \textbf{[22]}).

\begin{lstlisting}[caption=Generator sederhana]
def countdown(n):
    while n >= 0:
        yield n
        n -= 1

for x in countdown(3):
    print(x)
\end{lstlisting}

\section{Anotasi tipe dan pemeriksa statis (Sub-CPMK 8.1, pengenalan)}

Python tidak mengharuskan pemeriksaan tipe di waktu jalan; alat seperti \texttt{mypy} membantu menemukan ketidakkonsistenan sebelum produksi. Contoh minimal pada fungsi yang sudah Anda tulis:

\begin{lstlisting}[caption=Anotasi tipe dasar]
def total(harga: list[float]) -> float:
    return sum(harga)

def main() -> None:
    print(total([10.0, 5.5]))
\end{lstlisting}

\section{Contoh tambahan: \texttt{itertools.islice} pada generator}

\noindent Kadang generator tidak perlu dikonversi penuh ke \texttt{list}. Fungsi \texttt{islice} mengambil potongan terbatas tanpa memuat seluruh deret ke memori; dokumentasi modul \href{https://docs.python.org/3/library/itertools.html}{\texttt{itertools}} pada pustaka standar (lampiran \textbf{[20]}) dan HOWTO gaya fungsional (lampiran \textbf{[22]}).

\begin{lstlisting}[caption=islice pada generator countdown]
from itertools import islice

def countdown(n: int):
    while n >= 0:
        yield n
        n -= 1

print(list(islice(countdown(10), 0, 4)))  # [10, 9, 8, 7]
\end{lstlisting}

\noindent Perintah contoh (di terminal, dari folder proyek): \texttt{mypy comprehension.py} atau berkas yang Anda kerjakan. Perbaiki tanda tipe jika muncul peringatan jelas (mis.\ tipe balik, daftar kosong).

\subsection{Kode: \texttt{comprehension.py}}
\introstudikasuskode
\begin{lstlisting}[language=Python,caption={\texttt{comprehension.py}}]
"""Contoh penggunaan list comprehension dan generator."""


def deret_genap(batas: int):
    """Generator yang menghasilkan bilangan genap hingga batas."""
    for i in range(batas + 1):
        if i % 2 == 0:
            yield i

def main() -> None:
    # List comprehension
    angka = [1, 2, 3, 4, 5]
    kuadrat = [x**2 for x in angka if x % 2 != 0]
    print(f"Kuadrat ganjil: {kuadrat}")
    
    # Generator
    genap = deret_genap(10)
    print("Deret genap 0-10:", list(genap))

if __name__ == "__main__":
    main()
\end{lstlisting}
