\section{Deskripsi dan Ruang Lingkup}

Modul praktikum ini adalah sumber belajar untuk mencapai \textbf{CPL}, \textbf{CPMK}, dan \textbf{Sub-CPMK} mata kuliah praktikum Python berbasis \textbf{Outcome-Based Education (OBE)}, selaras berkas \path{silabus_praktikum_python.tex} dan kerangka \path{struktur_modul_praktikum_OBE.text}. \section{Capaian Pembelajaran Lulusan (CPL) yang Dibebankan}

\begin{itemize}[leftmargin=*]
  \item \textbf{CPL-1 (Pengetahuan):} Menguasai konsep algoritme dan pemrograman prosedural serta prinsip modularitas, struktur data, penanganan data/berkas dalam konteks Python, dan \textbf{pengenalan} pola kerja kode yang mendekati praktik produksi (tipe statis, \textit{logging}, struktur paket).
  \item \textbf{CPL-2 (Keterampilan umum):} Menerapkan pemikiran logis, sistematis, dan kritis dalam merancang, mengimplementasikan, serta menguji program Python untuk masalah komputasi, termasuk skenario I/O dan tugas ringan \textbf{konkuren/asinkron} pada tingkat pengenalan.
  \item \textbf{CPL-3 (Keterampilan khusus):} Menulis, menguji, dan memperbaiki program Python mulai dari skrip dasar hingga struktur berorientasi objek sederhana, penggunaan pustaka standar, pengujian unit tingkat pengenalan, serta \textbf{pengenalan} manipulasi data numerik/tabular dengan pustaka ilmiah umum.
  \item \textbf{CPL-4 (Sikap):} Menunjukkan tanggung jawab di laboratorium, mematuhi K3 dan etika akademik (lisensi perangkat lunak, kejujuran tugas), serta mampu berkolaborasi saat tugas kelompok.
  \item \textbf{CPL-5 (Pengetahuan / profesionalisme praktis):} Memahami praktik pengelolaan lingkungan virtual, dependensi, kualitas kode (\textit{debugging}, dokumentasi singkat), \textbf{pengenalan} anotasi tipe dan alat pemeriksa, \textit{logging} terstruktur, serta kebijakan kemasan proyek kecil sebagai dasar kerja profesional.
\end{itemize}

\section{Capaian Pembelajaran Mata Kuliah (CPMK)}

\begin{itemize}[leftmargin=*]
  \item \textbf{CPMK-1:} Mahasiswa mampu menyiapkan lingkungan Python 3, memahami sintaks dasar, identifier, tipe data, operator, serta masukan/keluaran konsol.
  \item \textbf{CPMK-2:} Mahasiswa mampu merancang alur program dengan percabangan dan perulangan, termasuk pola bercabang dan penggunaan \texttt{match} pada tingkat pengenalan.
  \item \textbf{CPMK-3:} Mahasiswa mampu memecah program menjadi fungsi yang dapat digunakan kembali, memahami parameter, nilai balik, \texttt{*args}/\texttt{**kwargs}, \textit{lambda}, dan anotasi tipe dasar.
  \item \textbf{CPMK-4:} Mahasiswa mampu menggunakan struktur data (\texttt{list}, \texttt{tuple}, \texttt{dict}, \texttt{set}) dan pemrosesan string lanjut untuk manipulasi data.
  \item \textbf{CPMK-5:} Mahasiswa mampu membaca/menulis berkas teks, bekerja dengan JSON sederhana, dan menerapkan penanganan kesalahan dengan \texttt{try}/\texttt{except}/\texttt{else}/\texttt{finally}.
  \item \textbf{CPMK-6:} Mahasiswa mampu mengorganisasi kode dengan modul, mengelola paket menggunakan \texttt{pip}, menggunakan \texttt{venv}, serta melakukan \textit{debugging} dan penerapan gaya kode konsisten.
  \item \textbf{CPMK-7:} Mahasiswa mampu menerapkan konsep lanjut berupa kelas dan pewarisan dasar, \textit{comprehension}/\textit{generator}, serta pengujian unit tingkat pengenalan (\texttt{unittest} atau \texttt{pytest}).
  \item \textbf{CPMK-8:} Mahasiswa mampu pada tingkat \textbf{pengenalan} laboratorium: menerapkan anotasi tipe dan pemeriksaan statis dasar, \textit{logging} terstruktur, struktur paket sederhana berbasis \texttt{pyproject.toml}/\texttt{venv}, serta satu pola eksekusi konkuren ringan (\texttt{concurrent.futures} atau \texttt{asyncio}).
  \item \textbf{CPMK-9:} Mahasiswa mampu memuat dan memanipulasi data numerik/tabular sederhana menggunakan \texttt{numpy} dan \texttt{pandas} (operasi dasar pada satu \textit{dataset} kecil).
\end{itemize}

\section{Keselamatan Kerja, Etika, dan K3 di Laboratorium}

\begin{itemize}[leftmargin=*]
  \item Patuhi tata tertib laboratorium; jangan membawa makanan/minuman ke area peralatan sensitif.
  \item Periksa kabel, ventilasi, dan posisi duduk ergonomis; laporkan kerusakan perangkat ke petugas.
  \item \textbf{Etika perangkat lunak:} gunakan Python dan editor sesuai lisensi; tidak menyalin tugas tanpa atribusi (plagiarisme).
  \item \textbf{Data dan jaringan:} ikuti kebijakan prodi untuk instalasi paket (\texttt{pip}); hindari mengunggah kode pribadi orang lain.
\end{itemize}

\section{Petunjuk Penggunaan Modul dan Pengumpulan Tugas}

\begin{enumerate}[leftmargin=*]
  \item Baca kotak \textbf{Sub-CPMK} di setiap bab sebelum praktikum.
  \item Kerjakan \textbf{aktivitas} dan \textbf{latihan}; simpan artefak sesuai konvensi penamaan.
  \item Format pengumpulan (placeholder --- sesuaikan LMS prodi):
  \begin{itemize}
    \item Berkas kode: \texttt{NIM\_TIF105\_M\textit{nn}\_tugas.py}\newline atau notebook \texttt{NIM\_TIF105\_M\textit{nn}.ipynb}
    \item Laporan (jika diminta): PDF dengan cover identitas dan ringkasan hasil
    \item Batas waktu: sesuai pengumuman dosen per minggu
  \end{itemize}
  \item Gunakan \textbf{checklist} di akhir bab untuk swalai; rujuk \textbf{Lampiran} untuk rubrik dan glosarium.
\end{enumerate}

\section{Tujuan Modul Praktikum}

Setelah menyelesaikan rangkaian bab dan tugas, mahasiswa diharapkan mampu:

\begin{itemize}[leftmargin=*]
  \item Menyiapkan lingkungan Python 3 dan menulis program konsol atau Jupyter sesuai petunjuk laboratorium.
  \item Mengimplementasikan algoritme dengan percabangan, perulangan, dan fungsi untuk masalah sederhana hingga menengah.
  \item Menggunakan struktur data, string, berkas, JSON, modul, dan lingkungan virtual pada tugas praktis.
  \item Menerapkan OOP dasar, gaya idiomatik (\textit{comprehension}/\textit{generator}), dan tes unit pengenalan.
  \item Mendemonstrasikan pengenalan anotasi tipe, pemeriksa statis, \textit{logging}, kemasan proyek ringan, dan satu pola konkurensi/asinkron sesuai Sub-CPMK 8.x.
  \item Memuat dan memanipulasi data kecil dengan \texttt{numpy}/\texttt{pandas} pada tingkat pengenalan (Sub-CPMK 9.1).
\end{itemize}

\section{Checklist selaras SN-Dikti (redaksi modul)}

\begin{itemize}[leftmargin=*]
  \item \textbf{Standar isi:} materi relevan dengan capaian kurikulum dan perkembangan IPTEKS; tidak hanya prosedur tanpa capaian.
  \item \textbf{Standar proses:} pengalaman belajar mahasiswa jelas (student-centered); urutan bab selaras RPS.
  \item \textbf{Standar penilaian:} rubrik dan instrumen terhubung Sub-CPMK/CPMK; konsisten dengan RPS TIF-105.
  \item \textbf{Standar sarana/prasarana:} kebutuhan lab realistis; alternatif jika fasilitas terbatas ditetapkan prodi.
  \item \textbf{Mutu dokumen:} versi/tanggal revisi dicatat pada identitas modul; peninjauan berkala.
  \item \textbf{Dokumentasi OBE:} setiap bab menunjuk rantai Sub-CPMK $\rightarrow$ CPMK $\rightarrow$ CPL melalui kotak capaian, sesi, dan asesmen formatif.
\end{itemize}
