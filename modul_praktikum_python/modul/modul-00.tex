\section{Deskripsi dan Ruang Lingkup}

Modul praktikum ini adalah sumber belajar untuk mencapai \textbf{CPMK} dan \textbf{Sub-CPMK} mata kuliah praktikum Python berbasis \textbf{Outcome-Based Education (OBE)}. Cakupan materi: lingkungan Python 3, sintaks dasar, struktur kontrol, fungsi, struktur data, berkas dan JSON, modul serta \texttt{pip}/\texttt{venv}, debugging, dilanjutkan topik lanjut (OOP dasar, \textit{comprehension}/\textit{generator}, pengujian unit pengenalan). Setiap bab memetakan aktivitas, tugas, dan asesmen formatif ke capaian yang terukur.

\section{Capaian Pembelajaran Lulusan (CPL) yang Dibebankan}

\begin{itemize}[leftmargin=*]
  \item \textbf{CPL-1 (Pengetahuan):} Konsep algoritme, modularitas, struktur data, dan penanganan data/berkas dalam Python.
  \item \textbf{CPL-2 (Keterampilan umum):} Pemikiran logis, sistematis, dan kritis dalam merancang, mengimplementasikan, serta menguji program.
  \item \textbf{CPL-3 (Keterampilan khusus):} Menulis, menguji, dan memperbaiki program Python dari skrip dasar hingga OOP sederhana dan pengujian unit pengenalan.
  \item \textbf{CPL-4 (Sikap):} Tanggung jawab di laboratorium, K3, etika akademik, kolaborasi.
  \item \textbf{CPL-5 (Profesionalisme praktis):} Lingkungan virtual, dependensi, \textit{debugging}, dokumentasi singkat.
\end{itemize}

\section{Capaian Pembelajaran Mata Kuliah (CPMK)}

\begin{itemize}[leftmargin=*]
  \item \textbf{CPMK-1:} Lingkungan Python 3, sintaks dasar, identifier, tipe data, operator, I/O.
  \item \textbf{CPMK-2:} Percabangan dan perulangan (\texttt{match} pengenalan).
  \item \textbf{CPMK-3:} Fungsi, parameter, \texttt{*args}/\texttt{**kwargs}, \textit{lambda}, anotasi tipe dasar.
  \item \textbf{CPMK-4:} Struktur data dan pemrosesan string lanjut.
  \item \textbf{CPMK-5:} Berkas teks, JSON, penanganan kesalahan (\texttt{try}/\texttt{except}/\ldots).
  \item \textbf{CPMK-6:} Modul, \texttt{pip}, \texttt{venv}, debugging, gaya kode.
  \item \textbf{CPMK-7:} OOP dasar, \textit{comprehension}/\textit{generator}, \texttt{unittest}/\texttt{pytest} pengenalan.
\end{itemize}

\section{Keselamatan Kerja, Etika, dan K3 di Laboratorium}

\begin{itemize}[leftmargin=*]
  \item Patuhi tata tertib laboratorium; jangan membawa makanan/minuman ke area peralatan sensitif.
  \item Periksa kabel, ventilasi, dan posisi duduk ergonomis; laporkan kerusakan perangkat ke petugas.
  \item \textbf{Etika perangkat lunak:} gunakan Python dan editor sesuai lisensi; tidak menyalin tugas tanpa atribusi (plagiarisme).
  \item \textbf{Data dan jaringan:} ikuti kebijakan prodi untuk instalasi paket (\texttt{pip}); hindari mengunggah kode pribadi orang lain.
\end{itemize}

\section{Petunjuk Penggunaan Modul dan Pengumpulan Tugas}

\begin{enumerate}[leftmargin=*]
  \item Baca kotak \textbf{Sub-CPMK} di setiap bab sebelum praktikum.
  \item Kerjakan \textbf{aktivitas} dan \textbf{latihan}; simpan artefak sesuai konvensi penamaan.
  \item Format pengumpulan (placeholder --- sesuaikan LMS prodi):
  \begin{itemize}
    \item Berkas kode: \texttt{NIM\_TIF105\_M\textit{nn}\_tugas.py}\newline atau notebook \texttt{NIM\_TIF105\_M\textit{nn}.ipynb}
    \item Laporan (jika diminta): PDF dengan cover identitas dan ringkasan hasil
    \item Batas waktu: sesuai pengumuman dosen per minggu
  \end{itemize}
  \item Gunakan \textbf{checklist} di akhir bab untuk swalai; rujuk \textbf{Lampiran} untuk rubrik dan glosarium.
\end{enumerate}

\section{Tujuan Modul Praktikum}

Setelah menyelesaikan rangkaian bab dan tugas, mahasiswa diharapkan mampu:

\begin{itemize}[leftmargin=*]
  \item Menyiapkan lingkungan Python 3 dan menulis program konsol atau Jupyter sesuai petunjuk laboratorium.
  \item Mengimplementasikan algoritme dengan percabangan, perulangan, dan fungsi untuk masalah sederhana hingga menengah.
  \item Menggunakan struktur data, string, berkas, JSON, modul, dan lingkungan virtual pada tugas praktis.
  \item Menerapkan OOP dasar, gaya idiomatik (\textit{comprehension}/\textit{generator}), dan tes unit pengenalan.
\end{itemize}

\section{Keterkaitan dengan RPS Berbasis OBE}

Modul ini memetakan secara eksplisit ke \textbf{CPMK} dan \textbf{Sub-CPMK} pada RPS resmi TIF-105: setiap bab memuat kotak Sub-CPMK, informasi sesi, sarana/prasarana, prosedur kerja, materi, aktivitas, latihan, asesmen formatif, checklist; dilanjutkan (pada bab tertentu) contoh kode, referensi sesi, dan rangkuman sebagai penutup isi bab.

\section{Ringkasan Konteks Kurikulum OBE}

Dalam OBE, penilaian menekankan bukti kinerja (program benar, laporan jelas, refleksi) dan umpan balik berkelanjutan. Praktikum melatih pemecahan masalah, \textit{debugging}, dan kebiasaan menulis kode konsisten.
